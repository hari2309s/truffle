# truffle — web app

The main PWA for Truffle, a voice-first personal finance assistant.

## Stack

- **Next.js 14** (App Router)
- **Supabase** — auth & database
- **LangGraph** — agentic finance workflows
- **Gemini / Groq** — LLM backends
- **ChromaDB** — vector memory

## Getting started

```bash
pnpm install
pnpm dev
```
